Explanation:
As I reduce the size of the browser window, the row with the dessert names breaks first. This is because the dessert names are placed in columns with the class col-md, which are wider than the columns with the class col-sm-2 used for the letters. The col-md columns are responsive to a larger breakpoint, while the col-sm-2 columns are designed for smaller devices, allowing them to remain on one line longer as the viewport size decreases.
